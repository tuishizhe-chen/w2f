# presentation/

The project slide deck — a self-contained, single-file HTML presentation.

```
presentation/
├── w2f_presentation.html   open in any browser (←/→ or space to navigate, F = fullscreen)
└── assets/                 all images + a vendored MathJax build (offline-capable)
```

Just open `w2f_presentation.html` in a browser; everything it needs lives in
`assets/`, so it works offline (no CDN). Keyboard: **→ / Space** next, **←** back,
**F** fullscreen, **Home/End** jump.
